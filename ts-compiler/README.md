# ts-compiler



## Setup
To run this project, install it locally using npm:

```
$ npm i -g typescript
$ npm init
$ npm i --save-dev lite-server
add "start":"lite-server" under "scripts"

$ npm start

watch mode 

$ npx tsc app.js -w / --watch (realtime compiling)

For multifile realtime 

$tsc --init
$tsc -w
```
